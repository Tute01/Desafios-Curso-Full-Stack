# Desafíos - Bootcamp Full Stack Engineer

Este repositorio reúne los desafíos y ejercicios que realicé durante el Bootcamp Full Stack Engineer de EducaciónIT.

El programa tuvo una duración aproximada de 7 meses y estuvo dividido en distintas fases de aprendizaje, donde cada desafío permitía aplicar los conceptos vistos en clase y ganar experiencia práctica mediante la resolución de problemas reales.

## Fase 1 - Arquitectura Web

En esta etapa aprendí los fundamentos del desarrollo web. Trabajé principalmente con HTML y CSS, construyendo sitios web estáticos y aprendiendo conceptos de maquetación, diseño responsive y organización de estilos.

## Fase 2 - Desarrollo Front-End

En esta fase incorporé JavaScript para agregar interactividad a las aplicaciones. Aprendí a manipular el DOM, trabajar con eventos, validar formularios y desarrollar funcionalidades dinámicas del lado del cliente.

## Fase 3 - Desarrollo Back-End

Durante esta etapa aprendí a desarrollar aplicaciones del lado del servidor utilizando Node.js. También trabajé con APIs, bases de datos, autenticación de usuarios y herramientas utilizadas en proyectos Full Stack modernos.

## Sobre este repositorio

Cada carpeta contiene los desafíos realizados durante cada fase del bootcamp. El objetivo es documentar mi progreso y conservar los ejercicios que fui desarrollando a medida que avanzaba en la formación.
